# backend_aula01

Criando o Repositório Back-end

## clonar projeto

git clone https://github.com/allanandradeprofessor/backend_aula01


ou

## Criar pasta do projeto

```bash
mkdir fullstack-backend
cd fullstack-backend
```

---

# Inicializar projeto Node.js

```bash
npm init -y
```

---

# Instalar dependências

```bash
npm install express cors
```

---



# Executando o servidor

```bash
node server.js
```
